# 인하대 빅데이터 학회 인빅(inbig) 5기를 모집합니다.

▶ 인하대 빅데이터 분석 학회 '인빅' 입니다. <br>
▶ 저희 '인빅'은 데이터 분석 프로젝트 진행과 지속적인 커뮤니티 형성을 목표로 합니다.<br>
▶ Python과 Tableau를 기반으로 진행하고 있습니다.<br><br><br>

▶저희 '인빅'은 새로운 커리큘럼으로 진행 할 예정 입니다.<br><br><br>

# 기초스터디 단계(Basic)
Python 기반 <br>
팀당 4인 구성 <br>
2주차 ~ 10주차까지 팀스터디 진행, 12주차~ 13주차 데이터 실습 프로젝트 진행 (시험 기간 제외)<br>
Notion으로 스터디 내용 업로드 <br><br><br>


# 시각화 단계(Data Visualization)
태블로를 사용해 데이터 시각화 <br>
팀당 4인 구성 <br>
2주차 ~ 7주차까지 팀스터디 진행, 10주차~ 13주차 시각화 실습프로젝트 진행 (시험 기간 제외) <br>
격주로 팀 스터디 내용 세미나 발표 <br> <br> <br>


# 머신러닝 단계(ML)
파이썬 머신러닝 패키지 (Scikit-learn) 이용<br>
팀당 3인 구성<br>
2주차 ~ 7주차까지 팀스터디 진행, 10주차~ 13주차 머신러닝 실습프로젝트 진행 (시험 기간 제외)<br>
격주로 팀 스터디 내용 세미나 발표<br><br><br>

# 프로젝트 단계(Project)
Kaggle, DACON 등의 데이터를 활용한 팀별 자율 프로젝트 진행<br>
팀당 3-4인 구성<br>
프로젝트 최종 결과물 종강 세미나 발표<br>
(머신러닝, 딥러닝에 대한 기초적인 지식이 필요합니다.)<br><br><br>

시각화 및 머신러닝 단계는 팀 스터디와 팀 발표가 함께 진행 될 예정입니다.<br><br><br>

팀 활동에 적극적으로 참여해주실 분만 지원 부탁드립니다.<br><br><br>
