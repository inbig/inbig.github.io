<인하대 빅데이터 학회 인빅(inbig) 4기를 모집합니다.>

▶ 인하대 빅데이터 분석 학회 '인빅' 입니다. <br>
▶ 저희 '인빅'은 데이터 분석 프로젝트 진행과 지속적인 커뮤니티 형성을 목표로 합니다.<br>
▶ Python과 Tableau를 기반으로 진행하고 있습니다.<br>

▶저희 '인빅'은 이번 학기부터 단계 및 새로운 커리큘럼으로 진행 할 예정 입니다.<br>

> 기초스터디 단계(Basic)
- Python 기반
- 3인 4팀
- 2주차 ~ 11주차까지 팀스터디 진행, 12주차~ 13주차 데이터 실습 프로젝트 진행 (시험 기간 제외)
- 세미나 진행 시 스터디 진행 사항 브리핑, 문제 풀이 발표 및 질문 


> 머신러닝 단계(ML)
- 머신러닝 입문(4인 3팀) / 딥러닝 입문(미정) / 자연어(미정) 
- 파이썬 패키지(Scikit-learn 등) 이용
- 2주차 ~ 11주차까지 팀스터디 진행, 12주차~ 13주차 머신러닝 실습프로젝트 진행 (시험 기간 제외)

* 각 단계는 팀 스터디와 팀 발표가 함께 진행 될 예정입니다.

* 자세한 정보는 인빅 공식 블로그를 참고해주시기 바랍니다.
인빅 공식 블로그 : https://inbig.github.io
